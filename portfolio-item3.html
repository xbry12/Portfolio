Hello, Welcome to Countries API

by Bryant Pyram

You can access the project here.

This is a Countries API. The goal of this project was to use a UI pattern that I wanted to learn and an API. I had to build out the UI pattern from scratch and have it use the API that I chose, which in this case was a list of countries. The UI Pattern that I wanted to use was Modals. It was particularly challenging for me to use Modals on React, because I had to learn things in React that I had no familiarity with, such as ternary statements. I plan on adding more usability for the modals such as a description of each country using an API if possible. The main challenge was figuring out the CSS for the project.
